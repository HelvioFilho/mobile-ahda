## Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/HelvioFilho/mobile-ahda && cd mobile-ahda
```

Follow the steps below
```bash
# Install the dependencies
$ npm install

# Start the project
# Android
$ expo run:android 

# IOS
expo run:ios
```
